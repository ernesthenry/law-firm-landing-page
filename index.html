⚖️ Build a Professional Law Firm Website

Your Mission: Use CSS Flexbox to create a stunning, professional law firm landing page. This is your final project to demonstrate mastery of Flexbox layouts.

🚀 Project Setup Guide

Step 1: Download the Project Files

Save this HTML file to your computer as "law-firm-project.html"

Step 2: Open in Code Editor

Use VS Code, Sublime Text, or any code editor to open the file

Step 3: Find the Tasks

Look for CSS comments starting with /* TASK 1: */ through /* TASK 7: */

Step 4: Complete Each Task

Uncomment the CSS properties and implement the flexbox layouts as specified

Step 5: Test Responsiveness

Resize your browser window after each task to see the responsive design in action
